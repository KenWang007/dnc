### dnc简介

* dnc = .NET Core、dotnet Core简称，方便快速输入、促进新技术流行，例如SF、ELK、k8s等简称对技术流行的助力
* dnc是微软新一代主力编程平台，开源、免费、跨平台、轻量级、高性能，支持Linux、Docker、k8s等环境，适合开发微服务、云原生、大型互联网应用、全开源解决方案
